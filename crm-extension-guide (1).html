crm.etg.team

CRM ETG Tools

Chrome Extension v2.0 · Handover + Auto Prolongation + Open Tickets + Close Duplicates

Установка
1
Скачай и распакуй crm-extension.zip в любую папку на компьютере
2
Открой Chrome и перейди по адресу chrome://extensions/
3
Включи «Режим разработчика» — переключатель в правом верхнем углу
4
Нажми «Загрузить распакованное» и выбери папку crm-extension
5
Готово! Открой crm.etg.team и перезагрузи страницу (F5) — кнопки появятся справа
ℹ При обновлении расширения Скачай новый zip, распакуй поверх старой папки, затем зайди в chrome://extensions/ и нажми кнопку 🔄 у расширения CRM ETG Tools.
Возможности
🔁
Handover
Меняет Owner на CRM Bot и Pool на incidents_handover_day одним кликом.
⏱
Auto Prolongation
Автоматически жмёт Prolongation. Причина — цветные кнопки, настройка для каждого тикета отдельно. Вкладка мигает пока активно.
📂
Open Tickets
Открывает тикеты из списка в фоне. Авто-обновление данных и авто-открытие по таймеру.
🗂
Close Duplicates
Закрывает дублирующиеся вкладки тикетов — оставляет по одной на каждый ID.
Как пользоваться

🔁 Handover

1️⃣
Открой тикет который нужно передать
2️⃣
Нажми 🔁 Handover справа — скрипт сам откроет редактирование, поставит CRM Bot и incidents_handover_day, сохранит

⏱ Auto Prolongation

1️⃣
Открой тикет и выбери причину продления — 4 цветные кнопки:
TPP
Client
Hotel
Technical problem
Активная кнопка яркая, остальные приглушены
2️⃣
Нажми ⚪ AutoProlong OFF — станет 🟢 AutoProlong ON
3️⃣
Скрипт сам жмёт Prolongation, выбирает нужную причину и сохраняет. Вкладки можно свернуть
4️⃣
Пока AutoProlong активен — заголовок вкладки мигает между ⏱ AutoProlong ON и названием тикета. После продления мигает ✅ Prolonged! 5 секунд
5️⃣
Каждый тикет помнит свою причину и состояние ON/OFF отдельно даже после перезагрузки
6️⃣
Нажми 🔼 Hide чтобы скрыть панель

📂 Open Tickets

1️⃣
Перейди на страницу со списком тикетов и перезагрузи страницу (F5) — панель появится внизу справа
2️⃣
📂 Open New Tickets — открывает только тикеты которых нет в открытых вкладках
3️⃣
📂 Open All Tickets — открывает все тикеты из списка включая уже открытые
4️⃣
⏱ open (min) + ⚪ Auto OFF — автоматически открывает новые тикеты по таймеру
5️⃣
🔄 reload (min) + 🔄 Reload OFF — нажимает кнопку Update data по таймеру и открывает новые тикеты. Страница не перезагружается полностью
6️⃣
Все вкладки открываются в фоне — текущая страница не переключается

🗂 Close Duplicates

1️⃣
Нажми 🗂 Close Duplicates в панели списка тикетов
2️⃣
Скрипт проверяет все открытые вкладки crm.etg.team/tickets/* и находит дубликаты по ticket ID
3️⃣
Оставляет первую вкладку каждого тикета, остальные закрывает. Показывает сколько вкладок закрыл
ℹ Панель Open Tickets Появляется только на странице со списком тикетов. Если не видно — перезагрузи страницу (F5).
⚠ Браузер должен быть открыт Вкладки можно свернуть. Но если закрыть браузер — всё остановится.

Installation
1
Download and unzip crm-extension.zip to any folder on your computer
2
Open Chrome and navigate to chrome://extensions/
3
Enable Developer Mode — toggle in the top right corner
4
Click Load unpacked and select the crm-extension folder
5
Done! Open crm.etg.team and reload the page (F5) — buttons will appear on the right
ℹ Updating the extension Download the new zip, unpack over the old folder, then go to chrome://extensions/ and click 🔄 refresh on CRM ETG Tools.
Features
🔁
Handover
Sets Owner to CRM Bot and Pool to incidents_handover_day in one click.
⏱
Auto Prolongation
Auto-clicks Prolongation. Reason via colored buttons, saved per ticket. Tab title flashes while active.
📂
Open Tickets
Opens tickets in background tabs. Auto data refresh and auto-open on timer.
🗂
Close Duplicates
Closes duplicate ticket tabs — keeps one per ticket ID, closes the rest.
How to use

🔁 Handover

1️⃣
Open the ticket you want to hand over
2️⃣
Click 🔁 Handover on the right — script opens edit mode, sets CRM Bot and incidents_handover_day, then saves

⏱ Auto Prolongation

1️⃣
Open a ticket and pick a prolongation reason using the 4 colored buttons:
TPP
Client
Hotel
Technical problem
Active button is bright, others are dimmed
2️⃣
Click ⚪ AutoProlong OFF — it becomes 🟢 AutoProlong ON
3️⃣
Script watches for the Prolongation button, selects the right reason and saves. Tabs can be minimized
4️⃣
While AutoProlong is active — tab title flashes between ⏱ AutoProlong ON and the ticket name. After prolongation fires, flashes ✅ Prolonged! for 5 seconds
5️⃣
Each ticket remembers its own reason and ON/OFF state separately, even after page reload
6️⃣
Click 🔼 Hide to collapse the panel

📂 Open Tickets

1️⃣
Go to the ticket list page and reload (F5) — the panel appears at the bottom right
2️⃣
📂 Open New Tickets — opens only tickets not currently open in browser tabs
3️⃣
📂 Open All Tickets — opens all tickets including already open ones
4️⃣
⏱ open (min) + ⚪ Auto OFF — automatically opens new tickets on a timer
5️⃣
🔄 reload (min) + 🔄 Reload OFF — clicks Update data on a timer and opens new tickets. Page does not fully reload
6️⃣
All tabs open in the background — your current page stays focused

🗂 Close Duplicates

1️⃣
Click 🗂 Close Duplicates in the ticket list panel
2️⃣
Script scans all open crm.etg.team/tickets/* tabs and finds duplicates by ticket ID
3️⃣
Keeps the first tab for each ticket, closes the rest. Shows how many tabs were closed
ℹ Open Tickets panel Only appears on the ticket list page. If you don't see it — reload the page (F5).
⚠ Browser must stay open Tabs can be minimized. But if you close the browser — everything stops.
